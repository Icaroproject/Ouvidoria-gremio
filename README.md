# 📢 Ouvidoria do Grêmio Escolar - EEEP Dom Walfrido

Sistema web desenvolvido para permitir que alunos registrem manifestações como sugestões, elogios, reclamações e denúncias, contribuindo para a melhoria da vida escolar.

---

## 🚀 Funcionalidades

- Cadastro e login de usuários
- Recuperação de senha via e-mail
- Registro de manifestações
- Filtro por tipo, status e turma
- Área administrativa (ADM)
- Área "Minha conta" para o aluno
- Validação em tempo real dos campos
- Interface moderna e responsiva

---

## 🧠 Objetivo

O sistema foi criado com o objetivo de:

- Dar voz aos alunos
- Melhorar a comunicação com a gestão escolar
- Tornar o ambiente escolar mais organizado e justo
- Incentivar a participação estudantil

---

## 🛠️ Tecnologias utilizadas

- PHP (PDO)
- MySQL
- HTML5
- CSS3
- JavaScript
- PHPMailer (sem Composer)
- Bootstrap
---

## ⚙️ Como rodar o projeto

1. Instale o XAMPP
2. Coloque o projeto em: C:\xampp\htdocs\
3. crie uma pasta com o nome gremio_ouvidoria_final


4. Importe o banco de dados no phpMyAdmin
5. Inicie o Apache e MySQL
6. Acesse: http://localhost/gremio_ouvidoria_final
